1. Как отказаться от изменений в файле (файлах) находящийся в рабочем каталоге (work directory)?

Ответ 1: git checkout .

2.Как вернуть файл из индекса в рабочий каталог?

Ответ 2: git restore --staged nameFile или старым способом git reset HEAD~1 

3.Как отменить ошибочный commit ?

Ответ 3: git log - и копируем №комита до которого хотим откатитьсяю 
	   git revert HEAD + №комита.
